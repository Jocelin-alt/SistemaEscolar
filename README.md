# Sistema Escolar CRUD

Proyecto desarrollado con Flask y MongoDB para la administración de:

* Alumnos
* Maestros
* Materias

## Funciones

### Alumnos

* Guardar alumnos
* Buscar alumnos
* Modificar alumnos
* Eliminar alumnos
* Reporte de alumnos

### Maestros

* Guardar maestros
* Buscar maestros
* Modificar maestros
* Eliminar maestros
* Reporte de maestros
* Turno Matutino y Vespertino

### Materias

* Guardar materias
* Buscar materias
* Modificar materias
* Eliminar materias
* Reporte de materias
* Horas semanales

## Tecnologías utilizadas

* Python
* Flask
* MongoDB Atlas
* HTML
* GitHub
